## ゲームストーリー
宇宙で旅をしている3つの星の使者たち。  
とある有力な情報をつかんだあなたは、  
他の使者に情報が漏れないように星へ急ぐのであった。

## ゲームルール
最初に好きな色のボールを選びます。
左右のキーで移動します。
同じ色のボールに当たると加速します。  
違う色のボールに当たったり、壁に当たってしまうとゲームオーバーです。
